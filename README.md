# What Desert Diamond Shade Are You?

A stylish interactive quiz built with **HTML, CSS, and JavaScript** that matches users to one of six Desert Diamond shades based on mood, style, and personality.

## Features

- Interactive personality quiz
- 6 unique Desert Diamond shade results
- Direct shade picker option
- Animated diamond result display
- Copy-to-clipboard share text
- Single-file app ready for GitHub Pages
- Responsive design for desktop and mobile
